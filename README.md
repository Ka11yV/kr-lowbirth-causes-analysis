#  대한민국 저출산 현상 및 원인 분석 프로젝트

본 프로젝트는 대한민국 사회의 핵심 난제인 저출산 현상의 심각성을 통계적으로 분석하고, 그 근본적인 사회·경제적 원인을 탐구하는 것을 목표로 합니다. 특히, 합계출산율 시계열 데이터와 국민의 출산/양육 가치관 데이터를 교차 분석하여 문제 해결을 위한 시사점을 도출합니다.

## 1. 프로젝트 개요 및 분석 목표
- **주요 분석 대상:** 합계출산율, 연령별 인구 구성비, 출산/결혼 의향, 사교육비 등
- **기술스택** Python (Pandas, Plotly, Streamlit)
